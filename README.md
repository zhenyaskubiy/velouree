# VELOURÉE - Online Cosmetics Store

## Project Overview
**VELOURÉE** is a modern, elegant, and fully responsive e-commerce website for a luxury cosmetics brand.  
The website showcases skincare, makeup, and beauty products with a **premium, feminine, and minimalistic design**, inspired by high-end beauty brands such as Dior Beauty, Glossier, and Rare Beauty.

The main goals of this project are:

- Provide a **smooth, conversion-focused shopping experience**
- Present products in a **visually appealing and brand-consistent manner**
- Ensure **mobile-first responsiveness** and accessibility
- Implement **clear calls-to-action** to drive sales

---

## Concept
**Brand Identity:**  
VELOURÉE represents **luxury, elegance, and modern femininity**. The website design reflects this identity through:

- Soft pastel color palette (ivory, blush pink, beige, soft gold accents)
- Elegant serif headings with clean sans-serif body text
- High-quality product imagery
- Minimalistic layout with high whitespace
- Smooth micro-interactions, hover effects, and subtle animations

**Website Features:**

1. **Navigation Menu:** Sticky header with logo, main menu, and icons for search, user account, wishlist, and shopping cart.  
2. **Hero Section:** Full-width banner with tagline **“Reveal Your Natural Glow”** and primary/secondary CTA buttons.  
3. **Featured Categories:** Visual cards highlighting main product categories (Skincare, Makeup, Lip Care, Face Care, Beauty Tools).  
4. **Best Sellers:** Product grid with hover effects, ratings, quick add-to-cart, and wishlist functionality.  
5. **New Arrivals:** Horizontal product carousel showcasing the latest products.  
6. **Brand Story:** Editorial-style section presenting VELOURÉE’s values and philosophy.  
7. **Customer Reviews:** Testimonial slider with star ratings and verified purchase badges.  
8. **Blog / Beauty Tips:** Magazine-style article grid with skincare and makeup tips.  
9. **Newsletter Section:** Email subscription form to join the VELOURÉE Beauty Club.  
10. **Footer:** Minimal multi-column layout with links, social icons, and payment options.
